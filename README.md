# Clinic Appointment System

## Overview
A full-stack web application developed as a university project to simplify clinic appointment booking and management.

## Features
- Patient registration
- Appointment booking
- Doctors listing
- Appointment management
- CRUD operations (Create, Read, Update, Delete)
- Database integration

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## My Role
I designed and developed the application, implemented the database, and integrated the front-end with the back-end.

## Learning Outcomes
This project enhanced my understanding of:
- Full-stack web development
- Database design and management
- PHP and MySQL integration
- Building responsive web applications
